Quick Start
1. Clone & Install
git clone https://github.com/your-org/deployboard cd deployboard npm install
2. Configure Environment
cp .env.example .env # Edit .env with your settings PORT=3001 RUNNER=local SITES_DIR=/var/www/user-sites TMP_DIR=/tmp/deployboard-builds MONGODB_URI=mongodb://localhost:27017/deployboard GITHUB_TOKEN=ghp_your_token_here
3. Start the Server
npm start # or for development: npm run dev
Docker Mode (VPS)
Docker mode requires a VPS with Docker installed. Not supported on Render's free tier.
Enable Docker Mode
# Add Docker socket permissions sudo usermod -aG docker $USER # Set runner mode RUNNER=docker
Build with Docker Compose
docker-compose up -d
Architecture
Frontend (this dashboard) — Pure HTML/CSS/JS with Xterm.js for the terminal and Socket.io for real-time log streaming.
Orchestrator (server.js) — Express.js API that receives deploy requests, manages the queue, and streams logs via Socket.io.
buildRunner.js — Abstraction layer that switches between Local (child_process) and Docker mode automatically based on RUNNER env var.
Nginx — Wildcard subdomain routing maps *.domain.com to /var/www/user-sites/$subdomain/dist
MongoDB — Stores project configurations, deployment history and environment variables.
